🌱 ECOMART – Eco-Conscious E-Commerce Platform

A data-driven e-commerce platform designed to promote sustainable purchasing decisions by evaluating products based on environmental impact and user behavior.

🚀 Overview

ECOMART is built to help users make eco-friendly shopping choices by integrating:

Sustainability scoring
Intelligent recommendations
Semantic product discovery
Automated data pipelines

The platform combines machine learning, backend engineering, and automation to create a smarter and greener shopping experience.

✨ Key Features

🟢 Eco-Score Recommendation System
Assigns sustainability scores to products using ML models
Built using LightGBM
Evaluates products based on behavioral and environmental signals

🔍 Semantic Product Search
Uses Nomic embeddings for intelligent search
Enables meaning-based product discovery instead of keyword matching

⚙️ Scalable Backend
Built with Node.js + Express
Structured data storage using PostgreSQL
Handles product, user, and recommendation data efficiently

🔄 Automated Data Pipelines
Integrated with n8n
Keeps product insights and sustainability metrics updated
Enables continuous data ingestion and processing
